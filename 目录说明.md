# 目录说明

## 项目层级

```text
repo-root/
  .agents/
  README.md
  CHANGELOG.md
  目录说明.md
  ocr-asr-evaluation.md
  plugins/
```

Codex marketplace 清单位于：`.agents/plugins/marketplace.json`

Codex 插件目录位于：`plugins/wx-video-account-notes/`

主要维护的 skill 目录位于：`plugins/wx-video-account-notes/skills/wx-video-account-notes/`

推荐使用 Vercel Labs 的 `skills` CLI 全局安装和管理该 skill：

```powershell
npm install -g skills
skills add https://github.com/MuZiJin701/wx-video-account-notes.git -g
```

全局安装便于多个 agent 复用同一份 skill。Codex 全局安装目标为 `~/.codex/skills/`，项目级安装目标为 `.agents/skills/`。如果使用 `cc-switch`，直接输入仓库地址即可。

## 文件职责

### `plugins/wx-video-account-notes/skills/wx-video-account-notes/SKILL.md`

skill 主说明，供模型执行时优先读取；这是仓库中主要维护的 skill 文件。

### `.agents/plugins/marketplace.json`

Codex marketplace 清单，声明当前仓库中的可安装插件条目。

### `plugins/wx-video-account-notes/.codex-plugin/`

Codex 插件元数据目录。

- `plugin.json`：Codex 插件清单

### `plugins/wx-video-account-notes/skills/`

仓库中的 skills 目录，Codex 插件也会读取这里的内容。

通过 `skills add` 或 `cc-switch` 安装管理时，使用仓库地址即可；工具会识别其中的 `wx-video-account-notes` skill。

### `scripts/`

PowerShell 入口脚本。

- `bootstrap.ps1`：准备 `.runtime` 目录（Python + venv + 依赖 + 模型文件）
- `invoke_pipeline.ps1`：执行主流程（下载→抽帧→OCR→ASR→材料→笔记）
- `verify_runtime.ps1`：检查运行时是否就绪

### `runtime/`

Python 源码。

- `resolve_link.py`：解析视频号分享链接
- `download_video.py`：下载视频
- `media_extract.py`：抽帧和抽音频
- `ocr_runner.py`：OCR（RapidOCR + 跳帧 + 提前退出优化）
- `asr_runner.py`：ASR（faster-whisper CPU）
- `compose_note.py`：生成 `note_materials.json`；无语音视频自动附加 `visual_frames`
- `pipeline.py`：主流程编排
- `bootstrap.py`：资产下载和安装

### `resources/`

模型成稿参考文件。

- `model-note-template.md`

### `.runtime/`

自动生成的项目内运行环境，包含工具（ffmpeg）、模型（faster-whisper tiny）、Python 虚拟环境和缓存。

### `tests/`

测试说明和 Python 单元测试。

### `examples/`

使用示例。

### `ocr-asr-evaluation.md`

OCR / ASR 实测记录与历史对比。
